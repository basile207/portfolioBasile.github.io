Système de recommandations de musique

Description

Création, analyse et mise en place d'une base de données adaptés au projet puis développement d'un système de recommandation musicale intelligent utilisant des algorithmes de machine learning. Le système analyse les préférences utilisateur, les historiques d'écoute et les caractéristiques des morceaux pour proposer des suggestions personnalisées.

Objectifs

  • Créer un algorithme de recommandation basé sur des données utilisateur ainsi qu'un large jeu de données
  • Analyser les features audio des morceaux (tempo, tonalité, énergie)
  • Développer une interface utilisateur intuitive

Technologies utilisées & Niveau d'utilisation

Python 95%
Scikit-learn 80%
Pandas 80%
NumPy 60%
PostgreSQL 50%

Méthodologie

Le projet a été réalisé avec des sprints de trois semaines et en équipe. Tout d'abord par la collecte, l'analyse et le nettoyage des données, puis le développement de plusieurs modèles de recommandation (user-based, item-based, hybrid) avant de les comparer et de les rendres plus performant.

Résultats

  • Base de données de 100 000 morceaux analysés
  • Temps de réponse moyen de 3 secondes
  • Les modèles sont optimisés et performants
  • Interface responsive testée sur différents appareils (à venir)

Développement et hébergement d'un site web dynamique

Description

Conception et développement complet d'un site web dynamique avec système d'authentification, gestion de contenu, base de données relationnelle et interface d'administration. Le projet inclut la mise en place de l'infrastructure d'hébergement et la configuration des services nécessaires.

Objectifs

  • Créer une application web moderne et responsive
  • Implémenter un système d'authentification sécurisé
  • Implémenter un système d'administration

Technologies utilisées & Niveau d'utilisation

HTML/CSS 95%
JavaScript 90%
PostgreSQL 80%
Docker 75%
MySQL 70%

Architecture

Lors de ce projet la méthodologie Agile a été utilisée pour structurer le développement. Au cours de sprint avec un scrum master désigné et un product owner pour définir les besoins.

Fonctionnalités principales

  • Système d'authentification avec refresh tokens
  • CRUD complet pour la gestion de contenu
  • Interface d'administration avec tableaux de bord
  • Upload et traitement de fichiers
  • Système de recherche et de filtres avancés
  • Responsive design adaptatif

Résultats

  • Temps de chargement < 2 secondes
  • Interface responsive
  • Sécurité validée
